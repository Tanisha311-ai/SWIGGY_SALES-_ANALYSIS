# SWIGGY_SALES-_ANALYSIS
Project Overview

This project focuses on analyzing Swiggy sales data using Python, Pandas, Plotly, and Power BI to uncover business insights related to revenue, customer ratings, food categories, geographical performance, and sales trends.

The dashboard provides an interactive and visual representation of key performance indicators (KPIs) that help stakeholders make data-driven decisions.

Objectives
Analyze overall sales performance.
Identify top revenue-generating cities and states.
Evaluate customer satisfaction through ratings.
Monitor monthly and quarterly sales trends.
Compare revenue contribution across food categories.
Create an interactive Power BI dashboard for business reporting.
